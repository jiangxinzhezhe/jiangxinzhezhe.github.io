Flex Container

this is a flex container (the blue area) with three flex items
The flex container becomes flexible by setting the display property to flex:
{display: flex;}

Flex Items

The direct child elements of a flex container automatically becomes flexible (flex) items.
The element above represents four blue flex items inside a grey flex container.

Flex Responsive

if you want to create a two-column layout for most screen sizes, and a one-column layout for small screen sizes (such as phones and tablets), you can change the flex-direction from row to column at a specific breakpoint (800px in the example)
@media (max-width: 800px) { .flex-container { flex-direction: column;}
